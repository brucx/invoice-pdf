## 三大跨语言通用意图（每个语言都出现）

1. **Word/Excel 格式词占比最大**（en 49 / id 55 / it 45 / de 44 / es 42 / fr 39 条）——
   用户默认心智是"下载 Word/Excel 模板"。机会：各页加"比 Word/Excel 模板好在哪"对比段
   （自动算税、不会错版式、手机能用），长期可做独立对比页。
2. **免费**（kostenlos/gratuit/gratis/무료/無料/miễn phí）——已是定位，确认出现在各页 title ✓。
3. **"英文/双语"变体在所有语言都存在且我们是唯一直接答案**：
   rechnungsvorlage englisch / modèle de facture en anglais / plantilla de factura en inglés /
   modello fattura in inglese / factuur maken engels / template invoice bahasa inggris /
   mẫu hóa đơn tiếng anh / 영문 인보이스 양식 (무료 다운로드) / 請求書 英語 テンプレート (無料) /
   invoice 範本 英文。→ **每个语言页加一个精确匹配的 H2 段**。
